Self Watering Plant System

A self-watering plant system using a Raspberry Pi would involve using the Raspberry Pi to control a watering system for plants, soil moisture sensors
and a pump system to move the water. The Raspberry Pi would be programmed to monitor the moisture level of the soil in the plant's pot and
automatically trigger the watering system when the soil becomes too dry.
The watering system consist of a small pump connected to a relay board which is then connected to the Raspberry pi,
which would pump water from a reservoir to the plant's pot.
The Raspberry Pi could also be programmed to monitor the temperature and light levels in the plant's
environment and adjust the watering schedule accordingly.
Overall, a self-watering plant system using a Raspberry Pi allows for efficient and automated plant care
